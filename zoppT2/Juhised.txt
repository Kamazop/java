Tunniülesanne 2.

Kõik ülesanded teha meetoditena, mille main-meetod etteantud
argumentidega välja kutsub. Pidage kinni meetodite spetsifikatsioonist.
Vastused väljastatakse konsoolile.

1. Olgu antud täisarvude massiiv. Anda vastuseks teise ja viimase liikme
korrutis. Kui massiivis on ainult 1 element, siis on vastuseks see element,
ja kui massiivi pikkus on 0, on vastuseks 0.
static int korrutis(int[]jada)
{2,3,5,6} -> 18
{8} -> 8
{} -> 0

2. On antud 2 sõnet. Kui esimene lõpeb sama sümboliga, millega teine
algab, on vastuseks nende ühend, kust on välja jäetud 1.sõne viimane
sümbol ja 2.sõne esimene sümbol. Muudel juhtudel jäetakse ühendist
välja ainult 2.sõne esimene sümbol. Kui 2.sõne on tühi, on vastuseks
esimene sõne.
static String kaksS6na(String s1,String s2)
"kapsas", "sibul" -> "kapsaibul"
"tere", "hommikust" -> "tereommikust”
"tere", "" -> "tere"

3. On antud täisarvude massiiv. Anda vastuseks tõeväärtus “true”, kui
selles leidub järjestikku kolm järjestikust arvu, ja “false” muidu.
static boolean kolmReas(int[] massiiv)
{2,3,3,4,4,2,3,4} -> true
{1,1,3,2,1,2,2,2} -> false
